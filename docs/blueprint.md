# **App Name**: AI Classroom Watch

## Core Features:

- Dashboard Overview: Provides a quick system overview with key performance indicators (KPIs) and visual representations of attendance data, with camera status warnings.
- Live Camera Monitoring: Displays live video streams from selected cameras, with overlays for classroom information (subject, time, etc.) and attendance controls. Utilizes an AI-powered tool to manage face recognition and automatically mark attendance.
- Lecture Management: Enables admins to create new lectures, schedule them for specific classrooms, and view/download attendance reports.
- Attendance Tracking: Allows filtering the data to check student attendance records.
- Student Management: Maintains student information, facilitating attendance monitoring and allowing for exports.
- Reporting and Analytics: Generates customizable reports (lecture-wise, student-wise, date range) with attendance summaries and export options.
- Defaulter Management: Identifies and manages students with low attendance records (overall, subject-wise, or date range).

## Style Guidelines:

- Primary color: Deep blue (#223142) for a professional and trustworthy feel, inspired by academic environments.
- Background color: Light gray (#F0F4F8), a subtle backdrop ensuring readability and focus on content.
- Accent color: Muted orange (#D98828) for highlights, call to actions, and important notifications, giving contrast but remaining professional.
- Body and headline font: 'Inter', a sans-serif font, for a clean, modern, and readable user interface.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent, professional icons throughout the app for navigation and actions.
- A clear, structured layout with consistent spacing to ensure ease of use.
